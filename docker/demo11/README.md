## Demo11 : Docker Hub 登入登出
當你註冊完 Docker Hub 帳號後，你可以嘗試使用語法進行登入。在這裡我們要教你如何使用語法在終端機上進行使用者登入與登出。

## 前置
需先擁有一組個人的 Docker Hub 帳號。若尚未註冊使用者，可以透過下方連結進行[註冊](https://hub.docker.com/)。

## 實作

### 登入

```
$ docker login

Username: cijie
Password:
```

> 上方範例的帳號密碼皆需由使用者自行更改為個人帳號與密碼

### 登出

```
$ docker logout
```
